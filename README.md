# 视频AI总结 - 通俗易懂的说明

> 把B站、小红书的长视频变成简洁的文字总结，就像给视频写了一份"懒人笔记"。

---

## 这个项目是做什么的？

想象你有一个视频，但视频太长了（可能1小时），你只想知道它讲了什么。

传统方式：你得耐着性子从头看到尾。

**有了这个工具**：你把链接扔进去，几分钟后，它会告诉你视频说了什么——就像有人帮你看了视频，然后给你一份笔记。

---

## 整体流程（就像做一道菜）

```
你（粘贴链接）
     │
     ▼
前端界面（点外卖的手机App）
     │ 把链接发给后台
     ▼
后台处理（厨房）
     │
     ├─ 第1步：下载视频 ──→ 从网上把视频"下载到本地"
     │
     ├─ 第2步：提取音频 ──→ 把视频的"声音"分离出来
     │
     ├─ 第3步：语音转文字 ──→ 把听到的话"写"成文字
     │
     └─ 第4步：AI总结 ──→ 让AI阅读文字，生成摘要
           │
           ▼
返回结果（外卖送到你手上）
```

**为什么需要这些步骤？**

| 步骤 | 比喻 |
|------|------|
| 下载视频 | 你不能隔着屏幕闻味道，必须把食材拿回家 |
| 提取音频 | 你不需要视频画面，只需要声音（就像电话） |
| 语音转文字 | 声音是"听到的"，但AI只能读"看到的文字" |
| AI总结 | 就像你让一个学霸朋友看完视频后给你讲重点 |

## 技术架构详解

### 1. 前端（你看到的网页）

**是什么**：一个漂亮的网页界面

**技术**：HTML + CSS + JavaScript（就是普通网页三件套）

**它做什么**：
- 显示输入框，让你粘贴视频链接
- 显示进度条，告诉你"现在在下载啦"→"现在在转写啦"
- 展示最终结果（AI总结的文字）
- 支持白天/黑夜两种皮肤切换

**类比**：就像外卖App的界面，展示菜单、显示订单状态、让你操作

---

### 2. 后端（看不见的"大脑"）

**是什么**：用Python写的服务端程序

**技术**：FastAPI + uvicorn

**它做什么**：
- 接收前端发来的视频链接
- 协调整个处理流程（调度各个模块）
- 保存处理结果（方便以后查看）

**类比**：就像外卖App的服务器，接收你的订单后告诉厨房做什么

---

### 3. 视频下载器（"采购员"）

**技术**：yt-dlp（一个开源的视频下载工具）

**它做什么**：
- 识别链接是B站还是小红书
- 用最适合的方式下载视频
- 报告下载进度（10%、20%、30%...）

**类比**：就像超市采购员，知道去哪个货架拿什么商品

---

### 4. 语音转文字（"速记员"）

**技术**：Whisper（OpenAI开源的语音识别模型）

**它做什么**：
- 接收音频文件
- 把人说的话转成文字
- 记录每个字在哪个时间点说的

**类比**：就像会议速记员，不仅记录说了什么，还标注"这段是第5分钟说的"

---

### 5. AI总结（"学霸朋友"）

**技术**：MiniMax大模型（国内AI服务）

**它做什么**：
- 阅读转写出来的文字
- 理解视频内容
- 生成简洁的总结

**类比**：就像你让一个看过无数视频的学霸朋友，帮你总结要点

## 数据存储结构

处理完成后，所有文件保存在 `outputs/{任务ID}/` 文件夹里：

```
outputs/
└── {随机任务ID}/
    ├── info.json        # 视频信息（标题、作者、时长）
    ├── transcript.txt   # 转写文字（原始文字稿）
    ├── segments.json    # 带时间戳的片段（每句话在哪个时间点）
    └── summary.txt      # AI生成的总结
```

**为什么要保存这些？**
- 你可以查看原始转写，确认AI总结得准不准
- 你可以下载字幕文件，在其他播放器里使用
- 以后想回顾，直接从历史记录里找

## 常见问题

### Q: 为什么下载很慢？
A: 可能是网络问题，或者视频本身很大。就像网购，国际包裹就是比国内慢。

### Q: 转写需要多久？
A: 大约视频时长的1/10到1/5。10分钟视频大概需要2-3分钟。

### Q: 支持哪些平台？
A: 目前支持B站和小红书。理论上yt-dlp支持的平台都可以添加。

### Q: 我的视频会被保存吗？
A: 视频文件只临时存储在内存中，处理完会删除。但转写和总结会保存在本地。

## 给外行的最终比喻

这个项目就像一个**自动化的视频笔记工厂**：

```
你扔进去一个视频链接
        │
        ▼
工厂自动运转：
  - 采购员拿回原材料（下载视频）
```

1. 复制视频链接
2. 粘贴到网页
3. 等待几分钟
4. 阅读AI生成的总结
